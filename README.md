# DriveEasy Car Rental Service

A responsive **frontend-only** prototype for a car rental booking system (BIT Web Technology assignment).

> This project intentionally contains **no backend** and **no database implementation**.  
> It simulates persistence and sessions using **browser localStorage** for academic demonstration.

## 🚀 Features
- Responsive design (Bootstrap 5 + custom CSS)
- 9 interconnected pages (Home, Fleet, Car Details, Booking, Dashboard, Profile, Login, Register, Contact)
- Dynamic fleet rendering + search and filtering (JavaScript/jQuery)
- Client-side form validation (Booking, Contact, Login, Register, Profile)
- Booking CRUD simulation (Create/Read/Update/Delete) using localStorage
- Auth/session simulation (register/login/logout) using localStorage

## 🛠 Installation
1. Clone the repository.
2. Open `src/index.html` in any modern web browser (Chrome/Edge recommended).
3. No server setup required.
